Modules#

BFModule#

The BFModule module consists of several sub-modules which provide interfaces for image acquisition, GenICam device control, CoaXPress register access, and CameraLink serial communication among others.

BufferAcquisition

BufferAcquisition Buffer Interface image acquisition library.

BFGTLUtils

BFGTLUtils Interface to access and control GenICam devices.

CXPReg

CXPReg Interface to read and write to the registry space of a CoaXPress camera.

CLComm

CLComm Interface to read and write to a Camera Link serial port.
